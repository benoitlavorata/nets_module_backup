User Access: Delete

In the standard UI enviroment the option "Delete" can be found under Action, this option is avilable for (all users)

After instaling this module the option "Delete" would be disable by default for all users (exaptate the admin).

This module allow the admin to enable or disable the "Delete" option from "Setting -> Users & Companies -> Users -> user naem (alex) -> Access Rights".
